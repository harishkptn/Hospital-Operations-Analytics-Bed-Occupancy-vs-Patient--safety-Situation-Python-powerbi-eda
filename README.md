# 🏥 Healthcare KPI Dashboard Analysis

## 🚀 Project Overview
This project analyzes hospital operational data to understand the relationship between **bed occupancy, staff responsiveness, and patient safety**.

The objective is to help healthcare management make **data-driven decisions** to improve patient outcomes and optimize staffing.

---

## 🧩 Problem Statement
Hospitals often operate under high occupancy levels, but the impact on **staff performance and patient safety** is not always clear.

Without insights:
- Patient safety risks may go unnoticed  
- Staff may be overburdened  
- Decision-making becomes reactive instead of proactive  

👉 This project identifies how occupancy impacts safety and performance.

---

## 🎯 Objectives
- Analyze hospital KPI data  
- Identify relationships between occupancy, responsiveness, and safety  
- Perform exploratory data analysis (EDA)  
- Build insights for better staffing decisions  
- Support management with actionable recommendations  

---

## 📁 Dataset Description
The dataset includes monthly healthcare KPIs:

- **Month** – Time period  
- **KPI1** – Average Bed Occupancy Rate  
- **KPI2** – Unassisted Fall Rate  
- **KPI3** – Staff Responsiveness Score  

---

## ⚙️ Methodology

### 🔹 Data Cleaning
- Checked duplicate records  
- Renamed columns for clarity  
- Dropped unnecessary columns  

### 🔹 Exploratory Data Analysis (EDA)
- Boxplots for outlier detection  
- Pairplot for relationship analysis  
- Heatmap for correlation  

### 🔹 Statistical Analysis
- Occupancy vs Staff Responsiveness → **-0.37 (negative correlation)**  
- Occupancy vs Fall Rates → **+0.70 (strong positive correlation)**  

---

## 🔄 Steps Involved
1. Data preprocessing using Python  
2. Handling missing values and duplicates  
3. Feature selection and cleaning  
4. Visualization using Seaborn & Matplotlib  
5. Correlation analysis  
6. Dashboard creation in Power BI  

---

## 🛠️ Tools & Technologies
- Python (Pandas, NumPy, Seaborn, Matplotlib)  
- Power BI (Dashboard & DAX)  
- Jupyter Notebook  

---

## 📈 Key Insights & Findings
- Higher bed occupancy leads to **increased patient fall rates**  
- Increased occupancy reduces **staff responsiveness**  
- Strong evidence of **patient safety risks under high load**
